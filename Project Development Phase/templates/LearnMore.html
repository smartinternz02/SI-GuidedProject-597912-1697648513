This web application utilizes Deep Learning techniques to classify Images based on the type of crime. A pre-trained model called Resnet50 is trained on a custom dataset to achieve the task


ResNet50 is a popular convolutional neural network (CNN) architecture used for image classification and feature extraction in deep learning. It is a variant of the ResNet (Residual Network) architecture, which was introduced by Kaiming He, Xiangyu Zhang, Shaoqing Ren, and Jian Sun in their 2015 paper titled "Deep Residual Learning for Image Recognition."



ResNet-50 is specifically known for its depth, featuring 50 layers of convolutional and fully connected layers. The key innovation in ResNet is the use of residual connections, also known as skip connections or shortcut connections. These connections allow the network to learn residual functions, making it easier to train very deep networks without encountering the vanishing gradient problem.


The ResNet-50 model consists of several building blocks, each of which includes multiple convolutional layers, batch normalization, and ReLU activation functions. Residual connections skip one or more of these blocks, allowing the model to directly learn the difference (residual) between the input and the output of the block.


Learn To Use The Application

This Application has its features divided among Common public users and users belonging to the Law Enforcement

The Home page contains two buttons, Law Enforcement, and Public, are available which take the user to the respective page

Common Public:

  • To Upload a pre-existing image from local storage, click on the Upload Button
  • To Click an Image using the Camera, click on the Click Picture Button. Capture, Retake and Done buttons will be available after clicking on Click Picture. To save the image, click on Done
  • Click on Classify to see the result
  • To select a new image, click on Back
  • To report to authorities, click on Report. (report button sends current time date and location details)

Law Enforcement:

  • To Upload a pre-existing image from local storage, click on the Upload Button
  • To Click an Image using the Camera, click on the Click Picture Button. Capture, Retake and Done buttons will be available after clicking on Click Picture. To save the image, click on Done
  • Click on Classify to see the result
  • To select a new image, click on Back
  • To print the result, click on Print (Print prints a PDF containing the image and result along with time and location. [ time and location are based on the device that generated the result and not on the image depicted])
